    Class StateTree

    一个 发布-订阅 模式的事件触发器类，相比于 EventSignal, EventInstigator 可以为特定事件添加多个回调函数

    相较于 EventSignal, EventInstigator 的删除事件性能较好，用于需要频繁删除监听器的情况，否则请使用 EventSignal
