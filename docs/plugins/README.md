**Plugins Overview**
- **Description:**: 插件集合文档，包含 `animSeq`, `fsm`, `ronin`, `stateTree` 等模块的详细 API、用法和示例。
- **Location:**: `src/engine/plugins/`

- **Modules:**
  - `animSeq`: 动画序列与层管理
  - `fsm`: 有限状态机（Finite State Machine）实现
  - `ronin`: 浪人核心扩展（玩家、控制器、HUD）
  - `stateTree`: 行为/动作状态树（任务 + 状态）
