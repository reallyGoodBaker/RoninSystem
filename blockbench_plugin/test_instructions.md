# Blockbench插件测试说明

## 插件安装步骤

### 方法一：直接安装（推荐）
1. 打开Blockbench
2. 进入 `文件` → `插件` → `安装插件`
3. 选择整个 `blockbench_plugin` 文件夹
4. 点击安装并重启Blockbench

### 方法二：手动安装
1. 将 `blockbench_plugin` 文件夹复制到Blockbench插件目录：
   - Windows: `%appdata%/Blockbench/plugins/`
   - macOS: `~/Library/Application Support/Blockbench/plugins/`
   - Linux: `~/.config/Blockbench/plugins/`
2. 重启Blockbench

## 测试步骤

### 1. 验证插件加载
1. 打开Blockbench开发者工具（F12）
2. 查看控制台是否有以下消息：
   ```
   Timeline Track Plugin 加载成功
   ```

### 2. 创建测试动画
1. 在Blockbench中创建一个新项目或打开现有项目
2. 进入 `动画` 模式
3. 创建一个新动画（如果没有的话）

### 3. 测试添加轨道功能
1. **通过菜单添加**：
   - 点击 `动画` 菜单
   - 选择 `添加自定义轨道`
   - 在弹出的对话框中选择轨道类型和名称
   - 点击确认

2. **通过工具栏添加**：
   - 在动画工具栏中找到 `添加轨道` 按钮
   - 点击按钮
   - 选择轨道类型和名称
   - 点击确认

### 4. 验证轨道添加
1. 检查时间轴中是否出现了新的轨道
2. 轨道应该有对应的颜色和名称
3. 应该看到成功消息：`已添加 [轨道名称]`

## 故障排除

### 如果插件没有显示：
- 检查插件是否正确安装
- 重启Blockbench
- 查看控制台错误信息

### 如果无法添加轨道：
- 确保已创建动画
- 检查是否在动画编辑模式下
- 查看控制台是否有错误信息

### 如果轨道不显示：
- 检查时间轴设置，确保所有轨道都可见
- 查看轨道是否被隐藏

## 预期结果

成功安装后，您应该能够：
- 在动画菜单中看到"添加自定义轨道"选项
- 在动画工具栏中看到"添加轨道"按钮
- 成功添加4种不同类型的轨道
- 在时间轴中看到新添加的轨道
- 收到添加成功的提示消息

## 调试信息

如果遇到问题，请在Blockbench开发者工具中检查：
- 控制台日志
- 错误信息
- 插件加载状态

插件会在控制台输出以下调试信息：
- `Timeline Track Plugin 加载成功`
- `轨道添加成功: [轨道数据]`
- 任何错误信息
